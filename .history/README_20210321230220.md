# project01
## 简介：
一个日程表，使用了vue.js，elementUI和indexedDB。
记录日程，每条事项可以记录标签，时间范围，提醒频率，备注，总结，地理位置，优先级等信息。
输入的事项信息会转换为对象放入indexDB中。
